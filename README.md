## Installation

### Curl

```
curl -fsSL https://opencode.ai/install | bash
```

### NPM

```
npm i -g opencode-ai@latest
bun i -g opencode-ai@latest
pnpm i -g opencode-ai@latest
yarn global add opencode-ai@latest
```

### Brew

```
brew install sst/tap/opencode
```

### AUR

```
paru -S opencode-bin
```
